# hakumei リリースノート

hakumei の主要な機能追加・修正の履歴です。アプリ内 (右上のベルアイコン) からも同じ内容を確認できます。

日付は JST。

## 2026-07-15

### コスト削減推奨を AWS 公式単価ベースに刷新 (サービス別分離・所在表示) [ACH-535]

**hakumei のコンセプトを強化する大改修**。推奨の削減見込み額に係数ハードコード ("ワンサイズダウン 30%" 等) を使わず、AWS 公式の料金 API から取得した実料金差ベースに全面刷新しました。

- **プライシング根拠を明示**: 推奨カードの「?」から `現行料金 $X → 変更後 $Y = Z% 削減 (東京、Price List API 取得日)` 形式で削減の根拠を確認できます。
  - S3 ライフサイクル: Standard `$0.025` → Standard-IA `$0.0138` (東京) の実料金差で削減額を算出 (44.8% 削減)
  - EC2 rightsize: 対象インスタンスの現行タイプと 1 段下タイプ (同ファミリ内) の Linux OD 実料金差で個別に集計
  - EBS 未使用: gp2 `$0.12` → gp3 `$0.096` (東京) の実単価差を reason に明示
- **根拠が算出できないカテゴリは「要個別評価」に統一**: NAT GW の時間料/処理料按分、Cross-AZ の集約可否、CloudFront のキャッシュヒット改善、CW Logs の保持期間短縮は、いずれもワークロード構成に依存して hakumei からは削減額を算出できません。楽観的な数字を出すのを止め、単価 (AWS Price List API) と現状月額のみを popover に明示、削減を実測するための運用ガイド (VPC Flow Logs で通信量測定、Compute Optimizer で使用率確認 等) を reason に記載する形に統一しました。
- **サービスカテゴリタブを EC2 と EBS に分離**: 従来の「EC2 / EBS」統合タブを「EC2」「EBS」の別タブに分割 (7 分類)。EC2 rightsize と EBS 未使用/gp2→gp3 が独立したビューになりました。
- **全ての推奨リソース行にアカウント名 + ID を表示**: 「アカウント: internal-tools (487220589982)」のように、組織全体ビューで各リソースの所属アカウントが一目でわかります。
- 副次バグ修正: EC2 リソースに「S3 Bucket」バッジが誤って付与されていた表示バグを修正 (根本原因: リソース ID 判定の正規表現が hex-only で demo モードの合成 ID を通過させず、S3 バケット判定に流れ込んでいた)。

## 2026-07-14

### 変化アラートを再設計 (同曜日ベースライン + アカウント別前月比) [ACH-529]

「表示件数が多くて形骸化している」「何と比べた差分か分かりにくい」というフィードバックへの対応。

- **判定基準を「期間内他日の平均 (Leave-One-Out)」から「同曜日の他日の中央値 (同曜日ベースライン)」に変更**しました。平日・休日の周期的変動が正常扱いになり、曜日起因の誤検知を構造的に解消。実データ確認で、旧仕様なら 7 日間で「連続上昇 30+ 件」が量産されていた期間が、新仕様で 1 件に絞り込まれました。
- **アカウント別・前月比の増減検知**を新設: 当月は日割り見込みで前月と比較し、大きく変動したアカウントを検知します (組織全体ビューかつ当月内 or 前月末のみ発火)。DB アップグレード等でコストが大きく増減したアカウントを早期に把握できます。
- **個票文言を「同曜日の平常値 $A を +$X (Y%) 上回りました」に変更**し、比較基準を明示。旧「$A → $B」矢印表現による「前日比」誤読を解消しました。
- **「連続上昇 (TREND_UP)」判定を廃止**: 持続的な超過は「コスト急増」カテゴリに自然に現れるため、独立ラベルは不要と判断しました。

- リロード後もセッションが継続するよう、会話 ID を sessionStorage に永続化しました。

### グラフ期間フィルターの不具合修正 [ACH-526]

- 根拠 popover で長いラベルがはみ出す表示バグを修正しました。

- 推奨の削減見込み額に「?」を追加し、計算根拠・参照データソース・参照期間を popover から確認できるようにしました。**ACH-535 (2026-07-15) で全カテゴリを AWS Price List API 実料金差に刷新した基盤です。**

- Name タグの取得と表示、タグフィルタの不具合を修正しました。
